Node.js / TypeScript
nexapay-node  ·  v1.0.0

Full TypeScript support. Works with Express, Fastify, Next.js, NestJS, and any Node.js environment.

npm install nexapay-node

🐍
Python
nexapay-python  ·  v1.0.0

Compatible with Django, Flask, FastAPI. Full async/await support with httpx.

pip install nexapay
View Docs Try Sandbox
🐘
PHP
nexapay-php  ·  v1.0.0

Works with Laravel, Symfony, WordPress plugins, or any plain PHP application. Requires PHP 8.0+.

composer require nexapay/nexapay-php

REST API / cURL
Any language  ·  HTTP

Use the raw REST API from any language — Ruby, Go, Java, Dart, or anything that can make HTTP requests.

curl -X POST \ -H "Authorization: Bearer npay_test_..." \ {{base_url}}/sandbox/v2/payments/charge
